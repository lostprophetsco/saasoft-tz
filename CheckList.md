# CheckList - Форма управления учетными записями

## Основные требования

### ✅ Стек технологий
- [x] Vue.js 3 + Composition API
- [x] TypeScript
- [x] Pinia
- [x] UI фреймворк (PrimeVue) - установлен и настроен

### ✅ Структура страницы
- [ ] Заголовок формы и кнопка добавления учетной записи (+)
- [ ] Панель с информацией и иконкой 
- [ ] Таблица со списком учетных записей

### ✅ Таблица со списком учетных записей
- [ ] Заголовки таблицы (Метки, Тип записи, Логин, Пароль)
- [ ] Ряды таблицы с учетными записями

### ✅ Ряд таблицы с учетной записью
- [ ] Метка: input text. Необязательное. Максимум 50 символов. В поле вводятся текстовые метки, через знак ;
- [ ] Тип записи: select. Обязательное. Значения: LDAP и Локальная
- [ ] Логин: input text. Обязательное. Максимум 100 символов.
- [ ] Пароль: password. Обязательное. Выводится при Тип записи = Локальная. Максимум 100 символов. Null по дефолту
- [ ] Кнопка удаления данной учетной записи.

### ✅ Логика работы
- [ ] При нажатии "+" в конец таблицы добавляется новая пустая запись без кнопки удалить
- [ ] При нажатии кнопки удаления запись полностью удаляется
- [ ] Валидация обязательных полей при потере фокуса
- [ ] Красная обводка для невалидных полей
- [ ] Сохранение/обновление при валидности

### ✅ Обработка данных
- [ ] Поле "Метка" преобразуется в массив объектов `{ text: "элемент метки" }`
- [ ] Разделитель меток - символ `;`
- [ ] Остальные поля сохраняются как текст
- [ ] Сохранение в Pinia store
- [ ] Сохранение в localStorage для персистентности
- [ ] Восстановление данных при перезагрузке страницы

### ✅ Особенности поведения
- [ ] Так как при создании учетной записи Тип записи не указан - поле Пароль не отображается и значение null
- [ ] При выборе "Локальная" поле "Пароль" показывается и сохраняется
- [ ] При выборе "LDAP" поле "Пароль" скрывается и сохраняется как null
- [ ] Динамическое отображение/скрытие поля пароля (colspan у логин?)
- [ ] Кнопка удалить в последней колонке отображается только у сохраненной учетной записи
- [ ] Если ни у одной учетной записи нет пароля - скрываем всю колонку, вместе с заголовком таблицы

## Дополнительные требования
- [ ] История коммитов в Git

- [ ] TypeScript типы для всех структур данных
- [ ] Компонентный подход

- [ ] Метка: максимум 50 символов (необязательная)
- [ ] Логин: обязательно, максимум 100 символов
- [ ] Пароль: обязательно для "Локальная", максимум 100 символов
